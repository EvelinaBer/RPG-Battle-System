#  RPG Battle System (C++)

A turn-based console RPG battle system where players can choose heroes and fight against enemies. The game features turn-priority based on Agility (AGI) and handles data through external JSON files.

## Features
- **Turn-Based Combat:** Actions are ordered by character Agility using a priority queue.
- **Classes:** Multiple roles (Tank, Healer, Attacker).
- **Data Driven:** Characters and enemies are loaded from `json` files.

## Technologies
- **Library:** [JSON for Modern C++](https://github.com/nlohmann/json) by nlohmann.
